# 📋 Product Brief A: HabitFlow — Daily Habit Tracker

## The Problem
People want to build better habits but lose motivation after a few days. Existing habit trackers are either too complex (10 screens of settings) or too simple (just a checklist). There's no middle ground that makes habit tracking feel satisfying and keeps streaks alive.

## The Product
**HabitFlow** — a clean, delightful daily habit tracker that focuses on streaks and momentum. One screen. No signup. No overwhelm.

## Target User
Young professionals (25-35) who've tried habit tracking apps before but always quit after a week. They want something that takes 30 seconds a day, not 5 minutes.

## Core Features
1. **Daily habit checklist** — Show today's habits with simple tap-to-complete checkboxes
2. **Streak counter** — Display current streak (consecutive days) with a fire emoji 🔥 that grows with streak length
3. **Add/remove habits** — Simple way to add a new habit (name + time estimate) or remove one
4. **Progress ring** — A circular progress indicator showing "3 of 5 habits done today"
5. **Motivational nudge** — A short encouraging message that changes based on progress ("Almost there!" / "Perfect day! 🎯")

## Design Direction
- Clean, minimal, lots of whitespace
- Warm color palette (think sunrise oranges and soft purples)
- Satisfying interactions — checkboxes should feel good to tap
- Mobile-first, but works on desktop too
- Vibe: "Calm productivity" — not gamified, not corporate

## Out of Scope (for this prototype)
- User accounts / login
- Backend / database (use localStorage or hardcoded data)
- Push notifications
- Weekly/monthly analytics
- Social features

## Success Criteria
A first-time user should be able to:
1. See their habits for today
2. Check one off
3. Feel the satisfaction of progress
...all within 5 seconds of opening the page.
