# Build Your Personal OS — AgVend GTM Team

An interactive, lesson-based course that teaches AgVend's go-to-market team how to build their own Personal OS inside Claude Co-work.

Your Personal OS is a set of files that makes Claude deeply understand who you are, what you do, and what matters to you — so every conversation starts with full context instead of a blank slate.

## Quick Start

1. Open the `personal-os/` folder
2. Follow the [lesson guide](personal-os/README.md)
3. Tell Claude Co-work: "Read 00-intro-cowork.md and teach me"

## What You'll Build

- **Constitution.md** — Your values, beliefs, and principles
- **Business.md** — Your role, customers, and competitive landscape
- **GOALS.md** — Priorities organized by timeframe and OKRs
- **AGENTS.md** — Instructions that make Claude remember everything
- **Tasks/** and **Knowledge/** — Your work items and reference material
- **BACKLOG.md** — A brain dump inbox that Claude processes into organized tasks

## Archive

The `archive/` folder contains the original Claude Code course materials (lessons, games, etc.) from the upstream fork. These are preserved for reference but are not part of the Personal OS course.
